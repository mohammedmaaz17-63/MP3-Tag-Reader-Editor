# MP3 Tag Reader & Editor in C

This project is a C-based MP3 Tag Reader & Editor developed to read and modify MP3 metadata using ID3 tags.

The project contains two implementations:

1. Command Line Argument (CLA) Version
2. Menu-Driven Version

Both implementations demonstrate binary file handling, dynamic memory management, and MP3 tag processing in C.
